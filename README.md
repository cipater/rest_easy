# RestEasy

Rest Easy: DRYs up your CRUD actions and resource finders.

## Installation

Add this line to your application's Gemfile:

    gem 'rest_easy'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install rest_easy
